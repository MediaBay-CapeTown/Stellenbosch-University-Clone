# Stellenbosch-University-Clone
This is a modern, responsive university departmental website for Stellenbosch University's Computer Science Division. Here's what defines it:  Core Identity A comprehensive institutional site showcasing academic programmes, research initiatives, people, and department news — bridging prospective students, current learners, alumni, and industry partners.  Technical Foundation Static HTML + Tailwind CSS with a custom global design system Vendor-prefixed flexbox for broad browser support (IE10+, Safari, Chrome, Firefox, Edge) Mobile-first responsive layout with fixed header/footer and scrollable content areas Cross-browser compatible resets and utilities Content Pillars Academics: undergraduate, postgraduate (honours/masters/phd), research groups, programme guides, modules People: staff profiles, students, alumni with rich biographical tables and images Engagement: detailed contact info (postal/physical addresses, emails, phones), interactive Google Maps, QR code News & Events: industry insights (Best Jobs 2024, 2025–2026 CS trends), 30+ annual events (hackathons, workshops, seminars, conferences) Institutional: about page, campus map, sustainability, accessibility, legal pages Design Language Consistent card-based UI with subtle shadows and hover effects Colour palette: primary navy (#1e3a6e), secondary gold (#fab52e), neutral grays Typography: Playfair Display headings, Inter body text Responsive grids: tables with alternating rows, multi-column layouts that adapt from mobile to desktop Key Features Added This Session ✅ Contact centre with SVG icons, coordinates, QR code ✅ Interactive Google Maps with JS fallback to iframe ✅ Career insights and 2025–2026 industry news ✅ 30+ events calendar ✅ Vendor-prefixed flex utilities for layout consistency  Bottom line: A professional, accessible, future-proofed departmental hub that works consistently across all modern browsers and devices.
