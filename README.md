# Stellenbosch-University-Clone
Stellenbosch University – Computer Science Division Website

A modern, responsive departmental website for the Computer Science Division at Stellenbosch University.

This platform serves as a comprehensive academic and institutional hub, connecting prospective students, current learners, alumni, researchers, and industry partners.

🎯 Core Identity

A professional, future-focused digital presence designed to:

Showcase academic programmes and research excellence

Highlight faculty, students, and alumni

Share departmental news, events, and industry insights

Provide clear communication and engagement channels

The site bridges academia and industry while maintaining institutional integrity and accessibility.

🛠 Technical Foundation

Built with performance, compatibility, and scalability in mind:

Static HTML + Tailwind CSS

Custom global design system

Vendor-prefixed Flexbox (IE10+, Safari, Chrome, Firefox, Edge support)

Mobile-first responsive architecture

Fixed header and footer with scrollable content areas

Cross-browser CSS resets and utilities

Progressive enhancement principles
