Undergraduate Modules

Overview of core and elective modules offered for the 2026 academic year.

BSc Computer Science (CS) Programme

Focal Areas

General CS

Combines Computer Science major with Science modules. See Section 4.4.1 of Science calendar.

Computer Systems

CS major with Electric and Electronic Engineering modules. See Section 4.4.2.

Data Science

CS major with Statistics and Data Science modules. See Section 4.4.3.

CS + Genetics

CS with Genetics as second major. See Section 4.4.4.

CS + GIT

CS with Geographical Information Technology. See Section 4.4.5.

Extended Programme

4-year extended curriculum with foundational support in Year 1.

First Year CS Modules

Code Module Name Semester Description
CS113 Computer Science for Actuarial Studies 1st Introduction to computer programming from a financial perspective.
CS114 Introductory Computer Science 1 1st Basic computer programming, data representation, and problem-solving.
CS144 Introductory Computer Science 2 2nd Data structures, algorithms, OOP principles including inheritance.

Second Year CS Modules

Code Module Name Semester Description
CSE214 Object-Oriented Programming 1st OOP principles, design patterns, testing, searching and sorting.
CS214 Data Structures and Algorithms 1st Classical data structures, algorithms, complexity analysis.
CS244 Computer Architecture 2nd Machine language, assembly, memory and process management.
SC272 Scientific Computing 1st & 2nd Linux, Python, Numpy numerical computing, plotting.

Third Year CS Modules

Code Module Name Semester Description
CS313 Computer Networks 1st Internet architecture, protocols, security, applications.
CS314 Concurrency 1st Concurrent systems, synchronisation, scheduling.
CS315 Machine Learning 1st Dimension reduction, ML techniques, neural networks.
CS343 Databases & Web Programming 2nd Relational databases, web services, cloud computing.
CS344 Program Design 2nd Specifications, reusable frameworks, testability.
CS345 Computability & Automata 2nd Formal languages, Turing machines, P vs NP.
SC372 Scientific Computing 1st & 2nd LaTeX, regex, Python Numpy & Scipy.

BDatSci Programme (4-Year Degree)

Focus: Computer Science (CS)

BDatSci with focal area Computer Science combines CS with Statistics and Data Science modules. See Section 4.6.1 of Science calendar for complete module list.

Fourth Year CS Modules for BDatSci

Code Module Name Semester Description
CS411 Computer Networks 1st Internet protocols, resource allocation, network security.
E414 Machine Learning 1st Dimension reduction, logistic regression, Gaussian mixtures.
CS412 Advanced Algorithms 1st Linear programming, approximation, randomised algorithms.
CS441 Machine Learning 2nd Advanced techniques and real-world applications.
CS471 Data Science Research 2nd Comprehensive project solving real-world data problems.
CS491 Space Science Algorithms 2nd Space science applications and algorithmic techniques.
CS495 Functional Programming N/A Introduction to functional programming paradigm.
